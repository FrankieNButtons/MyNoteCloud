# Openclaw 配置

## 服务配置

### 安装`Homebrew`

使用HomeBrew官方安装工具安装HomemeBrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 安装`Openclaw`

```bash
brew install openclaw
```

### 启动服务并开始配置

初次启动就会进入配置交互界面

```bash
openclaw onboard
```
